# Information Architecture

## 주요 화면 구조

### 1. 온보딩 (`/onboarding`)
- Step 1: 가용시간 설정
- Step 2: 버퍼 설정
- Step 3: 첫 프로젝트 생성
- Step 4: 첫 일정 배치 체험

### 2. 대시보드 (`/`)
- 프로젝트별 투입률
- 가용시간 대비 계획 총량
- 버퍼 소진율
- 알림/경고 배너
- 주요 지표 카드

### 3. 캘린더 (`/calendar`)
- 주/월 뷰
- 드래그앤드롭 일정 배치
- 긴급 테스크 빠른 추가
- 반복 일정 관리
- 버퍼 소진 시각화

### 4. 프로젝트 관리 (`/projects`)
- 프로젝트 목록
- 프로젝트 생성/수정
- 투입률 조정
- 프로젝트별 버퍼 설정

### 5. 타임라인/WBS (`/timeline`)
- 프로젝트별 작업 구조
- 간트 차트 형태
- 작업 진행 상황

### 6. 회고 (`/retrospective`)
- 주/월 회고 작성
- 계획 vs 실제 비교
- 예측 보정

### 7. 설정 (`/settings`)
- 가용시간 관리
- 버퍼 설정
- 알림 설정
- 데이터 관리 (내보내기)

### 8. 포모도로 (모달)
- 타이머 UI
- 작업 선택
- 세션 기록
