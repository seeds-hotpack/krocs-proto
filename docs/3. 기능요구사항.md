# 기능 요구사항

## 핵심 기능

### 1. 가용시간 및 버퍼 관리
- 주/월 단위 가용시간 설정
- 전역 버퍼 설정 (warn/block 정책)
- 프로젝트별 버퍼 설정

### 2. 프로젝트 관리
- 프로젝트 CRUD
- 프로젝트별 투입률 계산 및 시각화
- 우선순위 설정

### 3. 테스크 관리
- 테스크 CRUD
- 예상 소요시간 설정
- 우선순위 및 기한 설정
- 실제 소요시간 추적

### 4. 일정 배치 (캘린더)
- 드래그앤드롭으로 일정 배치
- 가용시간/버퍼 제약 실시간 검증
- 정책 위반 시 경고/차단
- 긴급 테스크 빠른 추가
- 반복 일정 관리

### 5. 포모도로 타이머
- 작업 집중 타이머
- 세션 기록
- 실제 수행 시간 자동 업데이트

### 6. 대시보드
- 프로젝트별 투입률
- 버퍼 소진율
- 일정 완료율
- 경고 알림

### 7. 타임라인/WBS
- 프로젝트별 작업 구조 시각화
- 진행 상황 확인

### 8. 회고 및 예측 보정
- 주/월 단위 회고 작성
- 예상 시간 vs 실제 시간 비교
- 다음 계획에 반영

### 9. 일정 재배치
- 일정 변경 시 자동 재배치 제안
- 여러 대안 제시

### 10. 알림 시스템
- 마감 임박 알림
- 버퍼 소진 경고
- 미배치 테스크 알림
- 주간 계획 리마인더

### 11. 온보딩
- 4단계 초기 설정 마법사
- 점진적 기능 노출

### 12. 데이터 관리
- Soft delete (30일 보관)
- 데이터 내보내기 (JSON)
- 자동 저장 (LocalStorage)
